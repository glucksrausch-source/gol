# Alchemic Conway's Game of Life

Ein hochperformantes, modifiziertes Conway's Game of Life in Python.
Dieses Projekt implementiert ein massives 2000x2000 Grid mit 10 erweiterten Regeln, einem Alchemie-System und einer scrollbaren Viewport-Kamera.

## Installation

Stelle sicher, dass Python 3.11+ installiert ist. Installiere die benötigten Abhängigkeiten:

```bash
pip install numpy scipy PySide6
```

## Ausführung

Starte das Programm über:

```bash
python3 src/main.py
```
